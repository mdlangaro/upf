2. Instalação

A instalação do PHSim é fácil e rápida. Sendo possível a escolha da linguagem usada para descrição dos passos.

 

O programa requer apenas 10240 KB de espaço disponível em disco. O local de instalação pode ser apontado no 3° passo.

 

O passo seguinte oferece opções para escolher o grupo onde serão criados os ícones do PHSim. Dando seqüência a esse passo o grupo padrão será PHSim 2.0.

 

Finalmente dá-se "próximo" para confirmar todas as informações passadas para o início da instalação.

Após o término da instalação, você está pronto para usar o Simulador PHSim. 2.0.
