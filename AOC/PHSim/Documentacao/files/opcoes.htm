4. Opções do PHSim

4.1. Preferências

Na opção visualizar do menu principal está localizada a ferramenta preferências onde se encontra as opções de editor e base numérica a ser trabalhada.

4.1.1. Editor

Esta ferramenta disponibiliza a troca da fonte de letra e de seu tamanho no ambiente desenvolvimento, tendo disponível todas as fontes instaladas no SO. Logo após alguma modificação na fonte, o editor oferece um preview da mudança.

4.1.2. Cores

Na aba cores, para uma melhor visualização, é possível modificar a cor dos determinados elementos do programa escrito em Assembly. Podendo também atribuir Negrito ou Itálico aos mesmos.

4.1.3. Simulação

A opção de base numérica trabalhada no ambiente simulação se localiza nessa aba. As duas bases disponíveis são decimal e hexadecimal.

4.2. Exportando para HTML

Uma ferramenta útil quando se trata de publicação do fonte do programa em Assembly é a Exportar para HTML, localizada na aba arquivo do menu principal. O arquivo é gerado no local especificado pelo usuário.
